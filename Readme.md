## gRPC Web API

### Data
- 上传数据集
    - post [用户名,数据集]
    - return [数据集状态]
- 查看临时数据集列表
    - get [用户名]
    - return [数据集列表]
- 查看某一个数据集
    - get [用户名,数据集id]
    - return [数据集]
- 删除临时数据集
    - get [用户名,数据集id]
    - return [状态]

### Model
- 模型训练
    - post [训练模式,数据集id]
    - return [实时状态,最佳模型]

- 模型评估
    - post [模型,数据集id]
    - return [实时状态,最终状态]

- 模型预测
    - post [模型,预测数据]
    - return [预测结果]
